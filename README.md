# markdown-image-relocate

Download and update images in Markdown files

## Usage

```bash
Usage: markdown-image-replace -f INPUT-FILE.md -o OUTPUT-FILE.md

Options:
      --version         Show version number  [boolean]
  -f, --read-markdown   Read a markdown file  [required]
  -o, --write-markdown  Write converved markdown file  [required]
  -d, --image-dir       Folder to store local images (default: ./img)
  -h, --help            Show help  [boolean]

Examples:
  Download and relocate images:  markdown-image-replace -f post.md -o converted.md
```
